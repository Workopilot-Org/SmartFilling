# 贡献指南

感谢您对本项目的关注！

## 贡献者许可协议（CLA）

由于本项目采用 **PolyForm Noncommercial License + 商业授权** 双授权模式，**所有外部贡献者必须签署贡献者许可协议（CLA）**，将您对贡献内容的版权和专利许可授予本项目维护者，以便维护者可以以本项目的许可条款（包括商业授权）分发您的贡献。

提交 PR 前，请通过以下方式获取并签署 CLA：

- 联系邮箱：`zheng.yang@workopilot.com`

## 如何贡献

1. Fork 本仓库
2. 创建您的特性分支（`git checkout -b feature/amazing-feature`）
3. 提交您的更改（`git commit -m 'feat: add amazing feature'`）
4. 推送到分支（`git push origin feature/amazing-feature`）
5. 创建一个 Pull Request

## 开发环境

- .NET 8.0 SDK
- Python 3.10+（验证码服务）
- 浏览器（Playwright 会自动下载）

## 代码规范

- 遵循项目现有的代码风格
- 为新增功能编写单元测试
- 确保所有测试通过：`dotnet test`

## 报告问题

请使用 GitHub Issues 报告 bug 或建议新功能。
